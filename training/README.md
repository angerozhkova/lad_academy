# Обучение модели RuModernBERT-base

Эта папка содержит материалы для дообучения модели `RuModernBERT-base` для бинарной классификации текстов.

## Содержимое
- `train_rumodernbert.ipynb`: Jupyter Notebook с кодом для дообучения модели.
- `hard_negative_dataset.csv`: Датасет с нецелевыми сообщениями с меткой 0 (формат: текст, метка).
- `real_label_1_dataset.csv`: Датасет с целевыми сообщениями с меткой 1 (формат: текст, метка).
- `syn_label_1_dataset.csv`: Датасет со сгенерированными сообщениями с меткой 1.
- `test.csv`: Датасет для тестирования без меток.

## Формат датасетов
- **Столбцы**: `text` (строка сообщения), `label` (0 или 1 для бинарной классификации).
- **Формат**: CSV, кодировка UTF-8, разделение ";".
